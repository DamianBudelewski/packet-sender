### Packet-sender
C program to send raw packets (eth->ipv4->tcp)

Type "./sendFrames -h" for possible parameters.

If you don't provide any parametrs then program will send raw TCP packet from lo interface to lo address.

### Parametrs:
*	a - acknowledgment number
*	b - interface
*	d - destination ip address
*	e - sequence number 
*	f - don't fragment 
*	g - fin flag 
*	h - help 
*	i - identification 
*	j - fragment offset
*	n - destination port 
*	o - type of service 
*	p - push flag 
*	q - ack flag 
*	r - rst flag 
*	s - source port 
*	t - time to live  
*	u - urg flag 
*	w - window size 
*	x - urgent pointer 
*	y - syn flag 
*	z - number of frames to send
